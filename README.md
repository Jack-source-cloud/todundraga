\# TodUndraga.mn 🌐



Corporate website built with Next.js + Tailwind CSS.



\## 🚀 Features



\* Mongolian 🇲🇳 / English 🇬🇧 language switch

\* SEO-friendly routing (/mn, /en)

\* Modern responsive design

\* Ready for CMS integration (Sanity)



\## 📦 Installation



```bash

npm install

npm run dev

```



\## 🌍 Deploy



Deploy easily on Vercel.



\## 🧠 Tech Stack



\* Next.js

\* Tailwind CSS
